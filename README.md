# Netflix Analysis

A data analysis project exploring Netflix Movies & TV Shows using Python, Pandas, and Matplotlib.

## Contents
- netflix_analysis.ipynb — Jupyter/Colab notebook with code, visualizations and insights.
- netflix_sample.csv — small sample of dataset for demo purposes.

## How to view
1. Open netflix_analysis.ipynb in Google Colab.
2. Run cells top to bottom. If prompted, upload netflix_sample.csv to the Colab file area.

## Tools & Libraries
- Python 3.x
- pandas, numpy, matplotlib, seaborn

## Key Findings (examples)
- Movies vs TV Shows distribution
- Top producing countries (US, UK, etc.)
- Growth of titles added by year
